Input
 - Binary Formate Process including PID (4B), Length of Sequence (4B), Reference Sequence (1B each)
 - Access Process with Reference Sequence

Output
 - Frames on PAS

 - 1L PT => 8 PTEs
 - 2L PT => 8 PTEs
